# AI Minecraft Indev

A browser-based recreation of Minecraft Infdev (early development version) built with Three.js.

## Features

- **Procedural World Generation** - Simplex noise-based terrain generation with Infdev-style world
- **Block Placement & Destruction** - Left-click to break blocks, right-click to place
- **Multiple Block Types** - Stone, Dirt, Grass, Wood, Glass, Sand, Ores, and more
- **First-Person Camera** - WASD movement with mouse look
- **Hotbar System** - Scroll wheel to switch between blocks
- **Home Screen Menu** - Create new worlds with custom seeds
- **Chunk Loading** - Dynamic chunk generation around the player

## Controls

| Key | Action |
|-----|--------|
| **W/A/S/D** | Move forward/left/backward/right |
| **Space** | Jump |
| **Mouse** | Look around |
| **Left Click** | Destroy block |
| **Right Click** | Place block |
| **Scroll Wheel** | Change selected block |
| **ESC** | Pause game |
